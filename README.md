Управление социальными проектами (Social Project Management)

Описание проекта

Данный проект представляет собой комплексную систему автоматизации управления жизненным циклом социальных проектов. Решение объединяет Telegram-бот для оперативного ввода данных и интерактивный аналитический дашборд для мониторинга социального эффекта и хода реализации инициатив.

Система позволяет организовать прозрачную работу с проектами: от регистрации идеи и назначения исполнителей до отслеживания статусов и визуализации ключевых метрик.

Цель проекта

Разработка инструментария для эффективного менеджмента социальных инициатив, обеспечивающего контроль сроков, прозрачность распределения задач и наглядную оценку социального импакта (Social Impact) в реальном времени.

Основные характеристики

Telegram-бот (Java): Полноценный интерфейс для управления проектами (создание, редактирование, удаление, смена статусов).

Аналитический дашборд (Python/Dash): Визуализация данных с графиками (Timeline, Pie Charts, Scatter Plots).

Единая база данных: Использование projects.csv для бесшовной синхронизации между ботом и аналитикой.

Мониторинг метрик: Отслеживание загрузки исполнителей, типов решаемых социальных проблем и соблюдения дедлайнов.

Автоматическое обновление: Дашборд считывает изменения из базы данных в режиме реального времени.

Предварительные требования

Java Development Kit (JDK) 17 или выше.

Python 3.8 или выше.

Maven (для сборки Java-бота).

Активный Telegram-бот (токен от @BotFather).

Установка и запуск

1. Настройка и запуск Telegram-бота (Java)

Бот является основным источником данных. Он создает файл projects.csv.

Откройте проект в IntelliJ IDEA.

Убедитесь, что в pom.xml добавлены зависимости telegrambots.

Запустите класс SocialProjectBot.java.

2. Запуск аналитического дашборда (Python)

Аналитика визуализирует данные, созданные ботом.

Установите необходимые библиотеки:

pip install dash pandas plotly


Запустите скрипт дашборда:

python dashboard.py


Откройте в браузере: http://127.0.0.1:8050/
